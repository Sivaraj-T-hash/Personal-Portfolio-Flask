# 📌 My Portfolio Website

This is my personal Portfolio Website built to showcase my profile, projects, achievements, and contact details. The website highlights my skills, certifications, and provides easy access to my resume and social links.

🔗 **Live Demo:** [https://sivaraj-portfolio.onrender.com](https://sivaraj-portfolio.onrender.com)

---

## 🚀 Features

* **📖 About Me:** Detailed section with profile, bio, and achievements.
* **📂 Projects:** Dynamic projects section with details and images.
* **🏅 Certifications:** Showcase of verified certificates from NPTEL and SkillRack.
* **📬 Contact Me:** Functional contact form integrated with Email.
* **🔗 Social Integration:** Quick links to LinkedIn, GitHub, Instagram, and SkillRack.
* **⚡ Responsive Design:** Optimized for a great viewing experience on both mobile & desktop.
* **🔐 Admin Panel:** A secure backend to manage and update projects easily.

---

## 🛠️ Tools & Technologies Used

### **Frontend**
* **HTML5** – For structuring the content.
* **CSS3** – For styling, layout, and responsive design.
* **JavaScript** – For interactions (toggle menu, typing effects, scroll animations).

### **Backend**
* **Python (Flask)** – For the Admin Panel, Login system, and dynamic content loading.
* **JSON Database** – To store project and certificate data dynamically.
* **Web3Forms** – To handle "Contact Me" emails easily.
